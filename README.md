# Red-AI-Scanner

## Quick Start

### Option 1: One-Click Start (Recommended)
You can start both the Backend and Frontend with a single script:

```bash
chmod +x start.sh
./start.sh
```

- Access Dashboard: [http://localhost:3000](http://localhost:3000)
- Backend API: [http://localhost:8000](http://localhost:8000)

Press `Ctrl+C` to stop both.

### Option 2: Manual Start

**Terminal 1: Backend**
```bash
cd backend
python3 main.py
```

**Terminal 2: Frontend**
```bash
cd frontend
npm run dev
```

## Troubleshooting
If scans fail, ensure the Backend is running the latest code by restarting it.
